# cop-3710WalmartRetailSalesAnalyticsDatabase

Design a large-scale retail analytics database using Walmart sales, economic indicators, holidays, and weather data. Must include time-based partitioning, rolling sales metrics using window functions, and triggers to log abnormal sales spikes.

Users:
Walmart Employees
Walmart Customers

Data sources: 
Store - the store number;
Date - the week of sales;
Weekly_Sales - sales for the given store;
Holiday_Flag - whether the week is a special holiday week 1 – Holiday week 0 – Non-holiday week;
Temperature - Temperature on the day of sale;
Fuel_Price - Cost of fuel in the region;
CPI – Prevailing consumer price index;
Unemployment - Prevailing unemployment rate;
Holiday Events (Christmas, Labor Day, Thanksgiving, etc., as well as the dates of such);

This project is planned to use the Oracle SQL Database application to compile its data.
